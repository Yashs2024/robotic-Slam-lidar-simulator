DH PARAMETERS
Denavit–Hartenberg Convention — Configure robot kinematics from specifications
T_i = R_z(θᵢ+θ_off) · T_z(dᵢ) · T_x(aᵢ) · R_x(αᵢ)
Each homogeneous transform encodes:
• θ_off — joint angle offset (deg)
• d — link offset along z (units)
• a — link length along x (units)
• α — twist about x (deg)
