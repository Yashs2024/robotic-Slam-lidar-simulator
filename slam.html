CAPABILITIES
Everything You Need to Learn Robotics
A professional-grade simulator packed with real-world SLAM algorithms, interactive controls, and stunning visualizations.
CORE SLAM
📡
LiDAR SENSOR
180° forward-facing LiDAR with adjustable ray density, sensor noise, and real-time distance measurements.
🧮
BAYESIAN GRID
Professional-grade occupancy mapping using Bayesian probability with gradient confidence shading.
📈
ODOMETRY DRIFT
Simulate real-world wheel encoder errors. The believed position diverges from ground truth.
🔗
LOOP CLOSURE
Scan-matching keyframe system detects revisited areas and corrects accumulated drift.
NAVIGATION & CONTROL
⚡
A* PATHFINDING
Heuristic shortest path through the occupancy grid with wall proximity penalties.
🔍
DIJKSTRA
Guaranteed optimal path without heuristics. Compare with A* to understand the tradeoff.
🐛
BUG2 WALL-FOLLOW
Reactive algorithm that follows walls to reach goals. Works without a map.
🧭
FRONTIER EXPLORE
Autonomous exploration detecting boundaries between known and unknown space.
🚗
3 DRIVE MODELS
Differential (tank), Ackermann (car-like), and Holonomic (omnidirectional) kinematics.
ADVANCED SYSTEMS
🎯
PARTICLE FILTER
Monte Carlo Localization with adjustable particle count. Watch hypotheses converge.
🔮
DYNAMIC OBSTACLES
Moving walls, sliding doors, and orbiting objects that challenge the SLAM system.
🌫️
FOG OF WAR
Unexplored areas remain hidden on the SLAM map, revealing only scanned regions.
TOOLS & VISUALIZATION
🌐
3D LIDAR CLOUD

📊
LIVE CHARTS

🏗️
MAP BUILDER
Draw custom walls or load presets — Maze, Warehouse, Office, L-Shape, Open Field.
📖
TUTORIAL SYSTEM
Guided 11-step walkthrough teaching every SLAM concept interactively.
